# Cursor AI Assistor

This is an OpenAI based code editor built for pair programming with AI. 

Feel free to file tickets for bugs or feature requests. Upvote 👍 the ones you'd like us to prioritize.

## Getting Started

### Setup

MacOS/Linux:

```
npm i
./setup.sh
```

Windows:

```
npm i
./setup.ps1
```

### Run

```
npm start
```
